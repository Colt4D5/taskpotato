# TaskPotato 🥔

A local-first time tracking application. No accounts. No backend. Just open and track.

## Features
- ⏱ Start/stop timers per task
- 💾 Persists in localStorage — survives tab close
- 📅 Tasks organized by day
- ✏️ Edit task details and manually adjust time
- 📊 Pie chart of time distribution
- 🖱️ Drag to reorder tasks
- ⌨️ Press N for new task
- 🔖 Live tab title with current task timer

## Tech Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Chart.js / react-chartjs-2
- Static export (GitHub Pages ready)

## Development

```bash
npm run dev
```
